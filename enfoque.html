Importancia del enfoque sistémico

El enfoque sistémico permite analizar y resolver problemas de manera integral.

Cita APA: Checkland, P. (1999). Systems Thinking, Systems Practice. Wiley.

Esta área de conocimiento (AC) proporciona una guía para la aplicación del enfoque de sistemas como medio para identificar y comprender problemas y oportunidades complejos, sintetizar posibles alternativas, analizar y seleccionar la mejor alternativa, implementar y aprobar una solución, así como implementar, utilizar y mantener soluciones de sistemas de ingeniería . La participación activa de las partes interesadas en todas las actividades del enfoque de sistemas es clave para su éxito.

En el contexto de sistemas de ingeniería, un enfoque de sistemas es un enfoque holístico que abarca toda la vida útil del sistema; sin embargo, suele aplicarse en las etapas de desarrollo y operación/soporte. Esta área de conocimiento define un enfoque de sistemas utilizando un lenguaje común y una base intelectual para garantizar que los conceptos, principios, patrones y herramientas prácticos de sistemas sean accesibles para realizar ingeniería de sistemas (IS).
